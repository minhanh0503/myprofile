## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/minhanh0503/myprofile.git
   ```

2. Navigate to the project directory:
   ```bash
   cd myprofile
   ```

3. Open `index.html` in your browser to view the portfolio locally.
